# **App Name**: PrimeEstate CRM

## Core Features:

- Unified Property Inventory: Comprehensive property management system allowing for sorting by status such as Active, Reserved, Pending, and Incomplete as requested.
- Advanced Parametric Filtering: Multi-layered search engine to filter properties by transaction type, price range, orientation, and specific architectural characteristics.
- Lead and Contact Ledger: A centralized database to manage clients with manual entry or directory-based selection workflows as specified in the UI blueprints.
- AI Listing Optimizer: An AI tool that uses property data to automatically generate high-converting, professional marketing descriptions for website listings.
- Dynamic Modal Operations: Fast-action modals for bulk updates, including property status changes, owner reassignments, and automated WhatsApp lead notifications.
- Professional PDF Generation: On-demand creation of property fact sheets with configurable privacy levels (confidential vs. public) for direct printing or export.
- Real-time Task Scheduler: An integrated calendar system for marking visits, setting task deadlines, and tracking real estate activity status in the database.

## Style Guidelines:

- Primary Color: Deep Executive Blue (#334659) to evoke trust and authority in real estate management.
- Accent Color: Vibrant Action Red (#E76556) for primary call-to-actions, status alerts, and floating action buttons.
- Background Color: Crisp Gallery White (#F4F6F8) providing high contrast for text readability and clean data presentation.
- Primary Font: 'Inter' (sans-serif) for its machined, modern aesthetic, ensuring numerical clarity across spreadsheets and property lists.
- Standardized FontAwesome 6 icon set for consistency in utility icons (homes, users, calendars, and currency).
- Stripe-style header with horizontal main navigation and specialized lateral 'Slide-in' modals for advanced filtering on the right-hand side.
- Subtle drawer-style slide-up transitions for mobile contact modals and lateral shifts for property filtering panes.
